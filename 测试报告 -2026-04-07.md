**脚本中的哈希函数：**
```javascript
function hashBun(s) {
  return Number(BigInt(Bun.hash(s)) & 0xffffffffn);
}

function hashFNV1a(s) {
  let h = 2166136261;
  for (let i = 0; i < s.length; i++) {

    h = Math.imul(h, 16777619);

```

✅ **结论：哈希函数一致**
- Bun 模式：`Bun.hash()` (wyhash)
- Node.js 模式：FNV-1a

## ⚠️ 常见问题排查

### 问题 1：宠物不匹配

**可能原因：**
1. 使用 Node.js 运行而不是 Bun
2. Windows 环境（使用 FNV-1a 哈希）
3. 脚本版本过旧

**解决方案：**
```bash
# 确保使用 Bun 运行
bun scripts/buddy-reroll.js --verify

# 检查运行环境
bun --version  # 应该输出 Bun 版本号
```

### 问题 2：找不到匹配的宠物

**可能原因：**
- 过滤条件太严格

**解决方案：**
```bash
# 降低属性要求
bun scripts/buddy-reroll.js --species capybara --rarity legendary --min-stats 50

# 增加迭代次数
bun scripts/buddy-reroll.js --species capybara --rarity legendary --max 10000000
```

### 问题 3：Windows 用户

**说明：** Windows 版 Claude Code（npm 安装）使用 FNV-1a 哈希，不是 wyhash。

**解决方案：**
```bash
# Windows 用户可以使用 Node.js 模式
node scripts/buddy-reroll.js --species capybara --rarity legendary
```
