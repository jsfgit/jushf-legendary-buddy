## 🚀 快速开始

### 方式一：双击启动

#### Windows
```
双击：启动工具.bat
```

#### macOS / Linux
```bash
chmod +x start.sh
./start.sh
```

然后按提示选择操作即可。

#### Windows (PowerShell)

```bash

bash scripts/buddy-helper.sh -a detect
```

## 🔧 环境要求

### 必需

- **Bun** (必须！Node.js 不行)
  - 安装：`curl -fsSL https://bun.sh/install | bash` (macOS/Linux)
  - 安装：`powershell -c "irm bun.sh/install.ps1|iex"` (Windows)
  - 验证：`bun --version`

### 可选

- **Claude Code** - 如果你想在 Claude 中使用宠物
  - 安装：`npm install -g @anthropic-ai/claude-code`

## 🔄 使用流程

### 普通用户

#### Windows
```powershell
# 1. 检测环境
powershell -ExecutionPolicy Bypass -File scripts/buddy-helper.ps1 -Action detect

# 2. 刷宠物
bun scripts/buddy-interactive.js --gallery
bun scripts/buddy-interactive.js dragon legendary 80

# 3. 写入配置
powershell -ExecutionPolicy Bypass -File scripts/buddy-helper.ps1 -Action write-uuid

# 4. 享受宠物
claude
/buddy
```

#### macOS / Linux
```bash
# 1. 检测环境
bash scripts/buddy-helper.sh -a detect

# 2. 刷宠物
bun scripts/buddy-interactive.js --gallery
bun scripts/buddy-interactive.js dragon legendary 80

# 3. 写入配置
bash scripts/buddy-helper.sh -a write-uuid

# 4. 享受宠物
claude
/buddy
```

### OAuth 用户

#### Windows (PowerShell)
```powershell
# 1. OAuth 配置向导
powershell -ExecutionPolicy Bypass -File scripts/buddy-helper.ps1 -Action oauth-setup

# 2. 按提示完成配置

# 3. 刷宠物并写入 userID
```

#### macOS / Linux (Bash)
```bash
# 1. OAuth 配置向导
bash scripts/buddy-helper.sh -a oauth-setup

# 2. 按提示完成配置

# 3. 刷宠物并写入 userID
```

## 💡 常见问题

### Q: 为什么必须用 Bun？
A: Claude Code 使用 Bun.hash() 算法，Node.js 的 FNV-1a 结果不匹配。

### Q: 传奇宠物很难刷吗？
A: 权重只有 1%，但配合脚本通常几百万次内能找到。

### Q: 属性多少算好？
A: 传奇宠物保底 50，建议 80+，90+ 为极品。

### Q: 闪光概率？
A: 1%，非常稀有，需要大量迭代。

### Q: macOS/Linux 可以用吗？
A: 可以！核心脚本完全跨平台，辅助脚本有 bash 版本支持。
